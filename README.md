# Morning PT

A simple 15-minute no-jump morning calisthenics timer built for men over 40.

## Features

- 15-minute timer
- 3 rounds
- 40 seconds work / 20 seconds rest
- Next workout prompt during rest
- Sound-only cues
- Sound on/off
- Test sound button
- Sound style selector
- No jumping
- iPhone and Android home-screen friendly

## Sound Cues

- Start: strong tone
- Rest: lower double tone
- Countdown: short beeps
- Finish: chime

## Sound Styles

- Classic Beep
- Drill Tone
- Soft Chime

## Troubleshooting Sound

- Tap **Test Sound** once before starting.
- On iPhone, turn off Silent Mode and raise media volume.
- On desktop, make sure the browser tab/site is not muted.
- If the old app is still showing, open the app link with `?v=6` at the end or remove and re-add the Home Screen icon.

## Standard Circuit

1. Incline Push-Ups
2. Bodyweight Squats
3. Forearm Plank
4. Reverse Lunges
5. Dead Bugs
